# プライバシーポリシー

最終更新日：2026年4月24日

## 1. はじめに

「1行日記」（以下「本アプリ」）をご利用いただきありがとうございます。本ポリシーは、本アプリが収集する情報とその利用方法について説明します。

## 2. 収集する情報

### 2-1. ユーザーが入力する情報
- 日記の本文（最大100文字）
- 気分（絵文字）
- 添付写真（プレミアム会員のみ。端末内にのみ保存され、クラウドには送信されません）

### 2-2. アカウント情報（ログイン時のみ）
- Google アカウントの表示名・メールアドレス（Google ログインを使用した場合）

### 2-3. 自動的に収集される情報
- アプリのエラーログ（端末内にのみ保存され、外部には送信されません）

本アプリは Firebase Crashlytics・Firebase Analytics などの外部アナリティクスサービスを使用していません。

## 3. 情報の利用目的

収集した情報は以下の目的にのみ使用します。

- 日記データのクラウドバックアップおよびデバイス間同期
- サブスクリプション状態の管理（RevenueCat）
- AI 分析機能の提供（プレミアム機能、Gemini API）

## 4. 第三者への情報提供

以下のサービスに必要な情報を提供します。いずれも広告目的での利用は行いません。

| サービス | 提供内容 | 用途 |
|---|---|---|
| Google Firebase | 日記の日付・本文・気分 + UID | バックアップ・認証 |
| RevenueCat | UID | サブスクリプション管理 |
| Google Gemini API | 日記の日付・本文のみ（氏名・メールアドレス等の個人識別情報を含まない） | AI 分析（プレミアム機能） |

## 5. データの保管

- **ローカルデータ**：端末内に SQLite データベースとして保存（日記、日記帳、設定）
- **クラウドデータ**：Google Firebase Firestore（東京リージョン）に保存（日記の日付・本文・気分のみ）
- **写真**：端末内にのみ保存。クラウドには送信されません
- **ロック用 PIN**：端末のセキュア領域（Android Keystore）にのみ保存。外部には送信されません
- **エラーログ**：端末内にのみ保存。外部には送信されません

## 6. データの削除

- 設定画面「アカウントとデータを削除」から、すべてのデータを完全に削除できます
- アカウント削除後、Firestore 上のデータは即時削除されます
- 端末内のデータ（写真を除く）も同時に削除されます

## 7. サブスクリプションの管理・解約

プレミアムプランの解約は以下の方法で行えます。

- **Android**：設定画面「プレミアムプランを管理」から Google Play のサブスクリプション管理画面へ移動して解約できます
- または Google Play アプリ → プロフィールアイコン → お支払いと定期購入 → 定期購入 から解約できます

## 8. 未成年者のプライバシー

本アプリは13歳未満の方を対象としていません。13歳未満の方からのデータ収集が判明した場合、速やかに削除します。

## 9. プライバシーポリシーの変更

本ポリシーを変更する場合、アプリ内またはこのページにて通知します。
